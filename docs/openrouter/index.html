#1API Router 全球口碑
01
STEP ONE

在 OpenRouter 注册并申请 API Key

打开 openrouter.ai 注册账号， 登录后点击首页 Get API Key 创建一个新的 API Key 并复制下来。

然后在顶部搜索框输入 free，会列出所有当前免费可用的模型。挑一个你想用的(比如最新的 deepseek / qwen / llama 系列)， 把它的模型 ID 也复制下来 —— 一般长这样:deepseek/deepseek-v4-flash:free。

注意:OpenRouter 的免费模型 ID 在结尾会有 :free 后缀, 粘贴时要带上这个后缀 —— 去掉就会变成付费版。
图 1 · OpenRouter 官网搜索 "free" 并申请 API Key
02
STEP TWO

在 EchoBird「模型中心」里添加 OpenRouter

打开 EchoBird，左侧进入 模型中心，在右侧栏切换到 模型中转站 标签 —— 第一个就是 OpenRouter。点击它旁边的 + 按钮。

在弹出的「编辑模型配置」窗口里:

① 名称 就是个标签，你能认出来就行，随便起。
② 模型 ID 才是真正传给 API 的字段，粘贴刚才复制的那个，例如 deepseek/deepseek-v4-flash:free，结尾的 :free 要保留 —— 带 :free 才会走免费版，去掉会变成付费版。
③ OpenAI 地址 已经自动填好，无需改动。
④ API 密钥 粘贴你在 OpenRouter 申请的那个 Key。
⑤ 点 保存。

提示:同一个 API Key 可以反复添加不同的模型，每个模型独立配置一行， 随时在「应用管理」、「我的 AI 项目」、「安装与修复」里切换使用。

#1API Router · global
01
STEP ONE

Sign up and grab an API key

Head to openrouter.ai, create an account, then click Get API Key on the homepage. Generate a key and copy it somewhere safe.

Type free in the search box at the top — every currently-free model will show up. Pick one you want to try (the newest deepseek / qwen / llama family is a good start) and copy its model ID. It looks like deepseek/deepseek-v4-flash:free.

Heads up: Free models on OpenRouter show a :free suffix — keep it when you paste (dropping it switches to the paid variant).
Fig 1 · Search "free" on OpenRouter and create an API Key
02
STEP TWO

Add OpenRouter inside EchoBird's Model Nexus

Open EchoBird, go to Model Nexus in the left sidebar, then switch the right-side pane to the Model Relays tab. OpenRouter is the first entry. Click the + button next to it.

In the "Edit Model Config" dialog:

① Name is just a label — type anything you'll recognise.
② Model ID is the field that gets sent to the API — paste the ID you copied, e.g. deepseek/deepseek-v4-flash:free, and keep the :free suffix — it selects the free variant; dropping it bills you for the paid one.
③ OpenAI URL is pre-filled — leave it alone.
④ API Key: paste the OpenRouter key.
⑤ Hit Save.

Tip: One key can back as many models as you want — add a separate row per model and switch freely from App Manager / My AI Projects / Install & Repair.
Fig 2 · Model Nexus → Model Relays → OpenRouter → hit +
Fig 3 · Model ID must keep the :free suffix
03
STEP THREE

Try it from Install & Repair

Jump to Install & Repair on the left, pick the OpenRouter model from the model selector at the bottom right, and send a quick message — free latency is genuinely fast. It is more than enough for everyday quick-fix prompts.

If you see a 429 / quota error, you have hit the 50-request daily cap — it resets next day. Or upgrade to Pay-as-you-go to bump it to 1,000/day.
